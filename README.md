# Tempus Romanum

An interactive clock visualization based on ancient Roman timekeeping.

## Description

In ancient Roman timekeeping, daytime (*dies* in Latin), the time between sunrise and sunset, was divided into 12 hours (*horae*) of equal length. The length of an hour changed throughout the year because of the earth's tilt as it revolved around the sun. Nighttime (*nox*) was divided into 4 parts of equal length called vigils (*vigiliae*). The morning was called *mane* and the evening was called *vesper*.

This clock visualization shows how Roman hours and vigils map onto the modern 24-hour system based on the sunrise and sunset in a certain place.

## Usage

```bash
npm install
npm run dev
```

Enter your local sunrise and sunset times and click Update to redraw the clock.
